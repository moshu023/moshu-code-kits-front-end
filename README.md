## 以vite2搭建的一个后台管理系统 界面模板


## 项目中使用的插件

* vue-router  路由管理
* vuex 状态管理
* vuex-persistedstate vuex持久化
* sass css预处理工具
* axios 网络传输
* element-plus ui框架
* echarts 图表插件
